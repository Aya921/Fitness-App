## âœ¨ Description
**Fitness App** is a **Flutter-based mobile application** designed to help users track and improve their fitness and nutrition.  

The app allows users to browse exercises, watch instructional videos, log their daily performance, and interact with an **AI-powered chatbot** for guidance and tips. Additionally, it provides **meal suggestions** and instructions on how to prepare them for a healthier lifestyle.  

It is built using **Clean Architecture** and **Bloc** for state management, ensuring scalable and maintainable code.

---


## ğŸ›  Features 

### **Authentication & Access**
- **Login / Signup:** Secure authentication with email and password  
- **Registration Details:** Users register with **gender, age, weight, height, goal, activity level**, and other personal info  
- **Forgot Password Flow:** Reset password via email and OTP verification  

### **Home & Dashboard**
- **Personalized Greeting:** "Hi [User Name], Let's Start Your Day"  
- **Category Selection:** Gym, Fitness, Yoga, Aerobics, Trainer  
- **Daily Recommendations:** Recommended exercises and muscles to focus on, e.g., Latissimus Dorsi, Quadriceps Femoris  
- **Upcoming Workouts:** Preview your scheduled workouts and see details  
- **Nutrition Suggestions:** Recommended meals for the day, e.g., Beef, Chicken, Dessert  
- **Explore Section:** Browse exercises, meals, or trainers for more options  

### **Fitness & Workout**
- **Search & Filter Exercises:** Find exercises by type, muscle group, or difficulty  
- **Exercise Details:** Complete instructions and videos for exercises  
- **Track Performance:** Log your daily workouts and monitor progress  

### **Chatbot**
- **AI Guidance:** Interact with an AI-powered chatbot for personalized fitness advice  
- **Exercise Tips:** Get real-time suggestions on exercises and form  

### **Profile & Settings**
- **Edit Profile:** Update personal information  
- **Change Language:** Switch between English and Arabic

## ğŸ›  Technologies Used

- **Flutter / Dart:** Cross-platform UI toolkit  
- **Bloc / Cubit (MVI Pattern):** Structured state management with clean separation of responsibilities  
- **Clean Architecture:** Ensures scalability, maintainability, and testability  
- **Repository Pattern:** Abstracts data sources for clean data management  
- **Dependency Injection:** Improves modularity and testability  
- **SonarQube:** Used for code quality analysis and maintainability checking 
- **Gemini:** Handles analytics, reporting, or other backend services
